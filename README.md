# AdminDex

A MERN stack admin dashboard application.
This application was done as a learning project.

Following project was finised by doing EdRoh tutorials.

## Features

- View data on separate tabs
- View data on dashboard 
- Search for data
- Filter data
- Sort data

## Technologies Used

- React
- Redux
- Material UI
- Nodejs
- Expressjs
- Mongoose
- MongoDB
